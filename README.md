# Calculadora Moderna
🧮 Calculadora Moderna

Calculadora moderna feita com HTML, CSS e JavaScript.

## 🚀 Funcionalidades

- Cálculo em tempo real
- Interface moderna estilo dark mode
- Botão CE (apagar último caractere)
- Botão C (limpar tudo)
- Suporte a teclado
- Resultado dinâmico enquanto digita

## 🖥️ Tecnologias

- HTML5
- CSS3 (Grid + Flexbox)
- JavaScript puro
